Libraries in the File Processing Suite
======================================

1. **file-processing**: The core library, provides file processing capabilities for over 20 formats.
2. **file-processing-ocr**: Extends `file-processing` with OCR capabilities.
3. **file-processing-transcription**: Enables transcription for audio and video files.
4. **file-processing-analytics**: Analyzes file collections and extracts metadata.
5. **file-processing-models**: Integrates machine learning models for processing.
6. **file-processing-indices**: Facilitates indexing for efficient data retrieval.
7. **file-processing-test-data**: Contains sample files for robust testing.
