Welcome to the File Processing Suite of Libraries Documentation
===============================================================

The **File Processing Suite** is a collection of Python libraries developed by the Data Science Team within the Office of the Chief Data Officer (OCDO) at Health Canada. This suite streamlines the processing, analysis, and indexing of diverse file types, integrating functionalities such as OCR, transcription, and analytics.
