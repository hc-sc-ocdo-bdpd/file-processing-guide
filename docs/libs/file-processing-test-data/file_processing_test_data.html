file_processing_test_data package

Module contents

The file_processing_test_data package provides sample test files for use in testing the file-processing suite of libraries. This package includes utilities to retrieve the path to test files and list all available test files.

Functions:

get_test_files_path: Returns the path to the directory containing test files. get_all_test_files: Returns a list of all test files in the test_files directory.

file_processing_test_data.get_test_files_path()[source]

Returns the path to the test_files directory.

This function locates the directory containing test files, which can be used for testing and development with the file-processing suite.

Returns:

Path object pointing to the test_files directory.

Return type:

Path

file_processing_test_data.get_all_test_files()[source]

Returns a list of all test files in the test_files directory.

This function gathers all files in the test_files directory, providing a list of file paths for use in automated tests.

Returns:

A list of file paths (as strings) for all files in the test_files directory.

Return type:

list
